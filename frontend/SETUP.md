## Create a New Project with Vite

    npm create vite@latest my-react-app
    cd my-react-app

    select: react framework
    select: language, in this case, JS

    cd react-app
    npm i
    npm run dev (This will start the server on the http://localhost:5173/.)

# Project Structure:

    node_modules
    public
    src
    .eslintrc.cjs
    .gitignore
    index.html
    package-lock.json
    package.json
    README.md
    vite.config.js
